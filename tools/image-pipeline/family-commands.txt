# BirdMap Image Pipeline Family Commands

Run one family block at a time when you want to collect more image candidates.
The collect command reuses cached candidates unless you add --refresh.

# Struthionidae (Ostriches)
npm run imagePipeline -- collect --family "Struthionidae (Ostriches)" --limit 8
npm run imagePipeline -- score --mock --family "Struthionidae (Ostriches)"
npm run imagePipeline -- generate-catalog --family "Struthionidae (Ostriches)"
